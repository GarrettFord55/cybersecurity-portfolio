# Networking Coursework

This folder contains certifications completed through the Cisco Networking Academy.

The courses provide foundational knowledge of networking infrastructure and configuration.

### Completed Courses

CCNA: Introduction to Networks

CCNA: Switching, Routing, and Wireless Essentials

### Skills Demonstrated

- OSI and TCP/IP networking models
- IPv4 and IPv6 addressing
- Ethernet switching
- VLAN configuration
- Router configuration
- Wireless networking fundamentals
- Basic network security concepts

These courses provided hands-on experience with networking technologies used in modern IT and cybersecurity environments.
