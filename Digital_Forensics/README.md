# Digital Forensics Investigations

This folder contains digital forensic investigation reports completed during coursework in Digital Forensics.

The investigations simulate real-world forensic analysis scenarios where digital evidence must be collected, analyzed, and documented.

### Skills Demonstrated

- Forensic disk imaging
- Artifact analysis
- Registry examination
- User activity reconstruction
- Evidence documentation and reporting

### Tools Used

- FTK Imager
- Forensic Explorer
- Axiom
- Windows forensic artifact analysis

These reports demonstrate the ability to perform structured forensic investigations and produce professional documentation of findings.
