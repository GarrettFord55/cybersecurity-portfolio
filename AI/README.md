# Artificial Intelligence Projects

This folder contains machine learning projects completed during coursework in an Introduction to Artificial Intelligence course.

These projects demonstrate the full machine learning workflow including:

- Data preprocessing and cleaning
- Feature preparation
- Model training
- Evaluation of prediction performance
- Interpretation of model results

The notebooks included here show the complete process of building predictive models using Python and Jupyter Notebook environments.

### Technologies Used

- Python
- Jupyter Notebook
- Machine Learning techniques
- Data preprocessing and analysis

These projects demonstrate how machine learning can be applied to real-world datasets to generate predictions and insights.
