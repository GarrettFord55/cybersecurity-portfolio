# Security+ Coursework

This folder contains coursework related to CompTIA Security+ concepts and labs.

The work focuses on fundamental cybersecurity principles and defensive security practices.

### Topics Covered

- Threat analysis and mitigation
- Network security
- Security architecture and design
- Risk management
- Malware and attack techniques
- System security practices

### Skills Demonstrated

- Identifying security threats and vulnerabilities
- Understanding defensive security strategies
- Applying security best practices in simulated environments

This coursework supports foundational knowledge required for cybersecurity roles and industry certifications.
